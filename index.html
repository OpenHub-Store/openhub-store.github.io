Features

Everything you need, nothing you don't

GitHub Store strips away the complexity of finding and installing open-source software.

Smart Discovery

Browse trending, recently updated, and new projects. Only repos with real installable binaries for your platform are shown — no source-only noise.

One-Click Installs

Tap "Install" and the right binary is downloaded and handed to your OS installer. APK on Android, EXE/MSI on Windows, DMG on macOS, AppImage/DEB on Linux.

Update Tracking

GitHub Store remembers what you installed and highlights when new releases drop. Stay up-to-date without checking GitHub manually.

Always Latest & Stable

Installs always come from the latest published, non-draft, non-prerelease version. The changelog you read is exactly what you're getting.

Open Source

Built with Kotlin Multiplatform and Compose. No ads, no tracking, no premium gates. Fork it, extend it, or contribute — it's Apache 2.0 licensed.

Rich App Details

Every listing shows stars, forks, rendered README, release notes with markdown formatting, and a full list of downloadable assets with sizes.

Get your app listed

GitHub Store uses the public GitHub Search API — no manual submission required. If your project meets these conditions, it appears automatically.

Read the Wiki
  • Public repository on GitHub
  • At least one published, non-draft release
  • Installable assets: .apk, .exe, .msi, .dmg, .pkg, .deb, .rpm, or .AppImage
  • Relevant topics and description for better discoverability
